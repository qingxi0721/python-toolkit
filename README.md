# pmtoolkit

* 日志类：支持配置日志输出等级、输出至stdout、stderr、输出为文件
* 数据校验类：以json格式编写数据校验文件
* 读取配置类：支持读取.ini、.xml、.yaml、.json格式文件并输出为字典
* 全局异常捕获类：创建全局异常捕获类后结合日志类进行全局异常捕获，支持设置日志等级。
